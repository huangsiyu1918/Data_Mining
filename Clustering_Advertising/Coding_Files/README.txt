Run Simple Random Algorithm:
1. Open the simple_randomized_algorithm.py File under python configuration environment
2. Modify the sample size, min support threshold in randomized_algo() function and modify dataset in reading data function
2. Type python simple_randomized_algorithm.py into the console to run the program

1. Open the SON_algorithm.py File under python configuration environment
2. Modify the min support threshold in randomized_algo() function and modify dataset in reading data function
2. Type python SON_algorithm.py into the console to run the program
